Scott–Vogelius

Click here to read what the information on this page means.

Orders\(1\leqslant k\)
Reference elementstriangle, tetrahedron
DefinitionThis is a mixed element containing these subelements:
  • order \(k\) vector Lagrange space
  • order \(k-1\) Lagrange space
Number of DOFstriangle: \((k+1)(3k+4)/2\)
tetrahedron: \((k+1)(k+2)(4k+9)/6\)
NotesThis is the same as the Taylor–Hood element, but the second subelement in this case is discontinuous between cells.
CategoriesMixed elements
