Dual polynomial

Click here to read what the information on this page means.

Abbreviated namesdual
Orders\(0\leqslant k\leqslant 1\)
Reference elementsdual polygon
Number of DOFsdual polygon(n): \(\begin{cases}1&k=0\\n&k=1\end{cases}\)
NotesThese elements are defined on the barycentric dual grid.
These elements are defined as a linear combination of Lagrange basis functions on the fine grid.
CategoriesScalar-valued elements

pip3 install symfem

import symfem

# Create dual polynomial order 0 on a dual polygon
element = symfem.create_element("dual polygon(4)", "dual", 0)

# Create dual polynomial order 1 on a dual polygon
element = symfem.create_element("dual polygon(4)", "dual", 1)
