Click here to read what the information on this page means.

Abbreviated namesBDM
Orders\(1\leqslant k\)
Reference elementstriangle, tetrahedron
Polynomial set\(\mathcal{P}_{k}^d\)
↓ Show polynomial set definitions ↓↑ Hide polynomial set definitions ↑
\(\mathcal{P}_k=\operatorname{span}\left\{\prod_{i=1}^dx_i^{p_i}\middle|\sum_{i=1}^dp_i\leqslant k\right\}\)
DOFsOn each facet: normal integral moments with an order \(k\) Lagrange space
On the interior of the reference element: integral moments with an order \(k-1\) Nédélec (first kind) space
Number of DOFstriangle: \((k+1)(k+2)\) (A002378)
tetrahedron: \((k+1)(k+2)(k+3)/2\) (A027480)
CategoriesVector-valued elements, H(div) conforming elements
