Brezzi–Douglas–Durán–Fortin

Click here to read what the information on this page means.

Abbreviated namesBDDF
Orders\(1\leqslant k\)
Reference elementshexahedron
Polynomial set\(\mathcal{P}_{k}^d \oplus \mathcal{Z}^{(3)}_{k} \oplus \mathcal{Z}^{(4)}_{k}\)
↓ Show polynomial set definitions ↓↑ Hide polynomial set definitions ↑
\(\mathcal{P}_k=\operatorname{span}\left\{\prod_{i=1}^dx_i^{p_i}\middle|\sum_{i=1}^dp_i\leqslant k\right\}\)

\(\mathcal{Z}^{(3)}_k=\operatorname{span}\left\{\nabla\times(0,0,x^{k+1}y), \nabla\times(0,xz^{k+1},0), \nabla\times(y^{k+1}z,0,0)\right\}\)

\(\mathcal{Z}^{(4)}_k=\left\{\nabla\times(0,0,xy^{i+1}z^{k-i}), \nabla\times(0,x^{i+1}y^{k-i}z,0), \nabla\times(x^{k-i}yz^{i+1},0,0)\middle|i=1,...,k\right\}\)
DOFsOn each facet: normal integral moments with an order \(k\) Lagrange space
On the interior of the reference element: integral moments with an order \(k-2\) vector Lagrange space
Number of DOFshexahedron: \((k+1)(k^2+5k+12)/2\)
CategoriesVector-valued elements, H(div) conforming elements
