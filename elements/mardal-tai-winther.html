Click here to read what the information on this page means.

Abbreviated namesMTW
Orders\(k=3\)
Reference elementstriangle, tetrahedron
Polynomial set\(\mathcal{Z}^{(7)}_{k}\) (triangle)
\(\mathcal{P}_{k}^d \oplus \mathcal{Z}^{(8)}_{k}\) (tetrahedron)
↓ Show polynomial set definitions ↓↑ Hide polynomial set definitions ↑
\(\mathcal{Z}^{(7)}_k=\left\{\boldsymbol{p}\in\mathcal{P}_{k}^d\middle|\operatorname{div}\boldsymbol{p}\text{ is constant, and }\boldsymbol{p}\cdot\boldsymbol{n}_{e_i}\text{ is linear on each edge }e_i\right\}\)

\(\mathcal{P}_k=\operatorname{span}\left\{\prod_{i=1}^dx_i^{p_i}\middle|\sum_{i=1}^dp_i\leqslant k\right\}\)

\(\mathcal{Z}^{(8)}_k=\left\{\nabla\times(xyz(1-x-y-z)\boldsymbol{p})\middle|\boldsymbol{p}\in\mathcal{P}_{k}^d\right\}\)
DOFsOn each facet (triangle): normal integral moments with an order \(1\) Lagrange space, and tangent integral moments with an order \(0\) Lagrange space

On each facet (tetrahedron): normal integral moments with an order \(1\) Lagrange space, and integral moments with an order \(1\) Nédélec (first kind) space
Number of DOFstriangle: \(9\)
tetrahedron: \(24\)
CategoriesVector-valued elements, H(div) conforming elements
