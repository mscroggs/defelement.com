Bernardi–Raugel

Abbreviated namesBR
Orders\(k=1\)
Reference elementstriangle, tetrahedron
Polynomial set\(\mathcal{P}_{k-1}^d \oplus \mathcal{Z}^{(10)}_{k}\) (triangle)
\(\mathcal{P}_{k-1}^d \oplus \mathcal{Z}^{(11)}_{k}\) (tetrahedron)
↓ Show polynomial set definitions ↓↑ Hide polynomial set definitions ↑
\(\mathcal{P}_k=\operatorname{span}\left\{\prod_{i=1}^dx_i^{p_i}\middle|\sum_{i=1}^dp_i\leqslant k\right\}\)

\(\mathcal{Z}^{(10)}_k=\left\{\left(\begin{array}{v}xy\\xy\end{array}\right),\left(\begin{array}{v}y(x+y)\\0\end{array},\left(\begin{array}{v}0\\x(x+y)\end{array}\right)\right)\right\}\)

\(\mathcal{Z}^{(11)}_k=\left\{\left(\begin{array}{v}xyz\\xyz\\xyz\end{array}\right),\left(\begin{array}{v}yz(x+y+z)\\0\\0\end{array},\left(\begin{array}{v}0\\xz(x+y+z)\\0\end{array}\right)\right),\left(\begin{array}{v}0\\0\\xy(x+y+z)\end{array}\right)\right)\right\}\)
DOFsOn each vertex: point evaluations in $d$ directions
On each facet: normal integral moments with an order \(k-1\) Lagrange space
Number of DOFstriangle: \(9\)
tetrahedron: \(16\)
CategoriesVector-valued elements, H(div) conforming elements
